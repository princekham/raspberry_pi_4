# raspberry_pi_4




to set static wireless lan ip address 
1. click the wifi icon,
2. advanced options
3. edit connections
4. IPV4 settings
5. change to 'Manual'
6. put ipaddress - and netmask 24 and then gateway and DNS - 172.26.1.1, 192.168.
